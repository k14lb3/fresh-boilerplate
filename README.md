# Fresh Boilerplate

Fresh boilerplate.

## Installation

```
git clone --depth=1 https://github.com/k14lb3/fresh-boilerplate.git <project-name>
cd <project-name>
rm -rf .git
git init .
```

## Usage

Install the dependencies:

```
deno install
```

Run the development server:

```
deno task start
```

Build for production:

```
deno task build
```
